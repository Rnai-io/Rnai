ส่วนที่ 1

📱 แอปมือถือ (rnai-mobile · Expo)

  • 15 ภาษา — คงของเดิม 8 ภาษา + เพิ่มอาเซียนครบ (Indonesian, Malay, Vietnamese, Filipino, Khmer, Lao, Burmese) ผ่าน shared language map
  • จำภาษาที่เลือก — บันทึกลง AsyncStorage จำค่าข้ามการปิดแอป
  • หน้านโยบาย/ข้อกำหนดในแอป — ฉบับทางการ TH/EN + หมายเหตุแปลในอีก 13 ภาษา (เลิกลิงก์ออกเว็บ)
  • เพิ่ม 2 สกิล (16 → 18) — Speech-to-Text (อัดเสียงจริงด้วย expo-audio) และ Extract Data (ข้อความ → JSON)
  • Failover ระดับแอป — สลับไป backend สำรองอัตโนมัติเมื่อหลักล่ม (EXPO_PUBLIC_API_FALLBACK_URL)
  • แก้การ์ดโมเดลให้ตรงจริง — เอาปุ่มดาวน์โหลดปลอมออก แสดงสถานะ cloud/Ollama ตามจริง
  • เลขเวอร์ชัน 2.0.0 + แก้ tsconfig ที่บล็อก typecheck

  • .env.example — เพิ่มตัวแปร SELF_* + เติมคีย์ infra ที่ขาด (Upstash/Stripe/Cloudinary/CRON)
✅ Deploy production แล้ว — https://rnai-io.vercel.app · /api/v1/status ตอบ operational · provider ครบ 4 ตัว
ส่วนที่ 3

🛡️ ระบบสำรอง Self-hosted (Modal)

โมเดลเปิดที่เรา host เอง เป็นชั้นสำรองสุดท้ายเมื่อผู้ให้บริการภายนอกล่ม — scale-to-zero (idle ไม่เสียเงิน)

serve (GPU)

vLLM · Qwen2.5-7B · OpenAI-compatible /v1/chat/completions — ครอบคลุมสกิลข้อความทั้งหมด

media (GPU)

SDXL (รูป) + Whisper (STT) + MMS (เสียง) รวมใน container เดียว — /generate /transcribe /tts

✅ Deploy ขึ้น Modal แล้ว · ทดสอบ /health = {"ok":true} · status backup ทั้ง 4 ช่อง configured:true · พิสูจน์ reachable:true เมื่อ container อุ่น

มีทางเลือกอื่นเตรียมไว้ด้วย: RunPod (official vLLM worker) และ docker-compose สำหรับ VPS/CPU

ส่วนที่ 4

🧠 ชุดสร้าง Rnai LLM ของตัวเอง

ครบวงจร fine-tune โมเดลแบรนด์ Rnai (ผู้ช่วยทั่วไป ไทย/อาเซียน 7–8B QLoRA)

  • แผน + base model — แนะนำ Qwen2.5-7B-Instruct (รองรับไทย, license เปิด)
  • seed dataset — ตัวอย่างโทนแบรนด์ + ไทย/อาเซียน + พฤติกรรมสกิล + การปฏิเสธอย่างสุภาพ
  • ขยายข้อมูลอัตโนมัติ — expand_dataset.py ใช้ Gemini สร้างหลักพันตัวอย่าง + กรองซ้ำ/ตรวจคุณภาพ
  • เทรนบนคลาวด์ — modal_train.py (QLoRA บน Modal GPU ไม่ต้องเช่าเครื่อง)
  • ประเมินผล — eval_rnai_llm.py + ชุดทดสอบ ใช้ Gemini เป็นกรรมการ (helpfulness/language/persona/safety + เทียบ base)

ผลที่เทรนเสร็จชื่อ rnai-llm เสียบเข้าระบบสำรองได้ทันที (SELF_VLLM_MODEL=rnai-llm)
